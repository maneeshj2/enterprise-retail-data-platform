# Enterprise Retail Data Platform

## Overview

This repository demonstrates the design and implementation of a production-style enterprise data platform.

The project simulates how operational systems expose REST APIs, which are consumed by a modern data engineering platform to build a scalable analytics solution.

## Technology Stack

- PostgreSQL
- FastAPI
- Docker
- Python
- AWS S3
- Snowflake
- dbt
- Apache Airflow
- Streamlit
